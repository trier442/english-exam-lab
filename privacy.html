개인정보처리방침

시행일: 2026년 7월 18일

1. 기본 원칙

영어시험연구소는 현재 회원가입, 댓글, 파일 업로드, 자체 문의 양식을 운영하지 않으며 사이트에서 이용자의 이름·전화번호·주소를 직접 수집하지 않습니다.

2. 자동으로 생성될 수 있는 정보

GitHub Pages와 같은 호스팅 사업자는 보안 및 서비스 운영을 위해 접속 시각, 브라우저 정보, IP 주소와 같은 기술 정보를 처리할 수 있습니다. 해당 정보의 처리는 각 서비스 제공자의 정책을 따릅니다.

3. 쿠키와 광고

현재 사이트 자체는 맞춤형 쿠키를 설정하지 않습니다. 향후 Google AdSense 또는 방문 통계 서비스를 도입하는 경우, 해당 사업자가 광고 제공과 성과 측정을 위해 쿠키 또는 유사 기술을 사용할 수 있습니다. 실제 도입 시 이 방침과 필요한 동의 절차를 업데이트합니다.

4. 외부 링크

사이트에는 EBS, GitHub 등 외부 사이트로 이동하는 링크가 포함될 수 있습니다. 외부 사이트의 개인정보 처리는 해당 운영자의 정책을 확인해 주세요.

5. 문의

개인정보 관련 문의는 문의 페이지의 안내에 따라 전달할 수 있습니다.

6. 변경 안내

수집 항목이나 외부 서비스가 변경되면 시행일과 변경 내용을 이 페이지에 반영합니다.
